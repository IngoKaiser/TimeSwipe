Detailed Description

Pseudo-Decorator class for a DAC: changing DAC behavior depending on CADmux's DACmode.

A pseudo-decorator class (just a simple wrapper class without a super-class) that contains pointers to the corresponding DACs and CADmux class and decorates 4 of DAC's methodes: SetVal, GetRealVal, SetRawOutput, GetRawBinVal which are required for a command processor
